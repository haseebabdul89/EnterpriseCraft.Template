EnterpriseCraft.Template is a production-ready ASP.NET Core starter template demonstrating how to build enterprise applications using Vertical Slice Architecture with a clear evolution path toward a Modular Monolith and eventually Microservices.

Rather than focusing solely on design patterns, this template emphasizes maintainability, modularity, scalability, and real-world engineering practices suitable for SaaS and enterprise solutions.

Included Features
1- Vertical Slice Architecture
2- CQRS with MediatR
3- ASP.NET Core Minimal APIs
4- Entity Framework Core
5- SQL Server
6- FluentValidation
7- Global Exception Handling
8- Modular Feature Organization
9- Feature-Based Endpoints
10-OpenAPI / Swagger
11-Dependency Injection by Module
12- Clean Project Structure
13- Production-Ready Foundation
Project Goals
Organize code around business capabilities, not technical layers.
Keep features isolated for easier maintenance and team collaboration.
Minimize coupling between modules.
Support incremental growth from a simple application to a large enterprise system.
Provide a reusable template for new ASP.NET Core projects.

Why This Template?

Many projects start with unnecessary complexity by adopting Microservices too early.

This template demonstrates an alternative approach:

Vertical Slice → Modular Monolith → Microservices

Build for today's business needs while preserving the flexibility to scale tomorrow.
